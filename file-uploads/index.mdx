## Features

- **Presigned URLs** - Direct client-to-S3 uploads without exposing credentials
- **Multi-file selection** - Select multiple photos with numbered indicators
- **Progress tracking** - Real-time upload progress in the UI
- **Storage quotas** - Per-user storage limits (configurable)
- **File validation** - MIME type and size validation server-side
- **Soft delete** - Maintain audit trail with soft deletes
