This guide covers the minimum tooling needed to run `apps/mobile` and
`apps/api` locally.

## Steps

## Required Tools

<Steps>
  <Step title="Node.js 20+">
    Install with nvm (recommended):
    ```bash
    nvm install 20
    nvm use 20
    node --version
    ```
  </Step>
  <Step title="pnpm">
    ```bash
    npm install -g pnpm
    pnpm --version
    ```
  </Step>
  <Step title="Expo CLI">
    ```bash
    npm install -g @expo/cli
    expo --version
    ```
  </Step>
  <Step title="Git">
    ```bash
    git --version
    ```
  </Step>
  <Step title="PostgreSQL">
    Use Docker or install PostgreSQL locally.
    See [Quickstart](/quickstart) for both options.
  </Step>
</Steps>

## Mobile Runtime Options

- **Expo Go (fastest)**: run on a physical device with the Expo Go app
- **iOS Simulator**: requires Xcode
- **Android Emulator**: requires Android Studio

## Troubleshooting

- **Node version mismatch**: run `nvm use`
- **pnpm not found**: reinstall with `npm install -g pnpm`
