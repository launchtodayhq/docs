# What's Included

Launch is a production-ready boilerplate. The code is complete‚Äîyou just need to connect your own accounts and API keys.

## How It Works

<Steps>
  <Step title="Clone the repo">
    All screens, components, and backend logic are ready to use.
  </Step>
  <Step title="Sign up for services">
    Create accounts with third-party providers (AWS, Stripe, etc.).
  </Step>
  <Step title="Add your API keys">
    Configure environment variables with your credentials.
  </Step>
  <Step title="Customize & ship">
    Modify screens, branding, and features to match your app.
  </Step>
</Steps>

## Feature Breakdown

### ‚úÖ Fully Built (Just Add API Keys)

| Feature             | What's Built                                          | You Need To                       |

## What You'll Customize

While everything works out of the box, you'll want to customize:

- **Branding** - Colors, logos, app name
- **Onboarding content** - Your value proposition
- **Home screen** - Your app's core functionality
- **Pricing** - Your subscription tiers
- **Copy** - Text throughout the app

<Tip>
  Start by running the app as-is to see all features working, then customize
  piece by piece.
</Tip>
