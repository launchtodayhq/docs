## Prerequisites

Before starting, ensure you have:

<Steps>
  <Step title="Node.js 20+">
    ```bash
    node --version
    ```
  </Step>
  <Step title="pnpm">
    ```bash
    npm install -g pnpm
    ```
  </Step>
  <Step title="Expo CLI">
    ```bash
    npm install -g @expo/cli
    ```
  </Step>
  <Step title="PostgreSQL">Either local install or Docker</Step>
</Steps>

## 1. Clone & Install

```bash
# Clone the repository
git clone https://github.com/your-repo/launch.git
cd Launch

# Install all dependencies
pnpm install
```

## 2. Set Up Database

<Tabs>
  <Tab title="Docker (Recommended)">
    ```bash
    cd apps/api
    docker-compose up -d
    ```
  </Tab>
  <Tab title="Local PostgreSQL">
    ```bash
    # macOS
    brew install postgresql
    brew services start postgresql

    # Create database
    createdb launch
    ```
  </Tab>
</Tabs>

## 3. Configure Environment

```bash
cd apps/api
cp env.example .env
```

Edit `.env` with your values:

```bash
# Database
DATABASE_URL="postgresql://postgres:postgres@localhost:5432/launch"

# Auth (generate a random string)
BETTER_AUTH_SECRET="your-super-secret-key-at-least-32-chars"
BETTER_AUTH_URL="http://localhost:3001"

# AI (optional)
# OPENAI_API_KEY="..."
# ANTHROPIC_API_KEY="..."

# File uploads (optional)
# S3_BUCKET="..."
# S3_REGION="..."
# S3_ACCESS_KEY_ID="..."
# S3_SECRET_ACCESS_KEY="..."

# CORS
TRUSTED_ORIGINS="http://localhost:3000,http://localhost:19006,launch://"
```

## 4. Run Migrations

```bash
cd apps/api
pnpm prisma generate
pnpm prisma migrate dev
```

## 5. Start the Backend

```bash
cd apps/api
pnpm dev
```

You should see:

```
ðŸš€ Server running on http://localhost:3001
```

## 6. Configure the Mobile App

The mobile app talks to the API via `EXPO_PUBLIC_API_URL`.

Set it in your Expo env (example):

```bash
# apps/mobile/.env
EXPO_PUBLIC_API_URL="http://localhost:3001"
```

If youâ€™re testing on a physical device, `localhost` wonâ€™t work. Use your machine LAN IP (e.g. `http://192.168.x.x:3001`) or an HTTPS tunnel.

## 7. Start the Mobile App

Open a new terminal:

```bash
cd apps/mobile
pnpm start
```

Then press:

- `i` for iOS Simulator
- `a` for Android Emulator
- Scan QR for physical device

## Common Issues

<AccordionGroup>
  <Accordion title="Can't connect to API from device">
    Update the API URL to your computer's local IP: ```bash # Find your IP
    ifconfig | grep "inet " | grep -v 127.0.0.1 ``` Then update
    `apps/mobile/.env` (`EXPO_PUBLIC_API_URL`) with your IP.
  </Accordion>

  <Accordion title="Database connection error">
    Ensure PostgreSQL is running: ```bash docker-compose up -d # or brew
    services start postgresql ```
  </Accordion>

  <Accordion title="Metro bundler error">
    Clear cache and restart: ```bash pnpm start --clear ```
  </Accordion>
</AccordionGroup>
