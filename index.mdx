## The Stack

Launch uses modern, battle-tested technologies that scale:

| Layer          | Technology          | Why                                          |
| -------------- | ------------------- | -------------------------------------------- |
| **Mobile**     | React Native + Expo | iOS, Android, iPad, and Android tablets      |
| **Navigation** | Expo Router         | File-based routing, deep linking built-in    |
| **UI**         | HeroUI + NativeWind | Beautiful components, Tailwind-style styling |
| **API**        | tRPC                | End-to-end type safety, no code generation   |
| **Database**   | Prisma + PostgreSQL | Type-safe queries, easy migrations           |
| **Auth**       | Better Auth         | Modern auth with social providers            |
| **Monorepo**   | Turborepo           | Fast builds, shared code                     |

Everything is TypeScript. Types flow from your database schema to your mobile UI. Change a field in Prisma, and TypeScript catches everywhere that needs updating.
